## 🧩 Project Overview
This project is a modern, responsive website for a coffee brand. It is designed with a **mobile-first approach** and optimized for viewing on mobile, tablet, and desktop devices. The layout adjusts smoothly across screen sizes, offering intuitive navigation and clean visuals through HTML structure, CSS styling, and JavaScript interactions.


---

## 🛠️ Technologies Used

- **HTML5** – Page structure and content  
- **CSS3** – Styling, layout, responsive design  
- **JavaScript** – Interactivity and dynamic behavior

The code is organized into:
- `index.html` → Main page layout  
- `style.css` → Visual styling and responsive media queries  
- `script.js` → Scroll, menu toggle, and interactive functionality  
- `images/` → Assets used in the design :contentReference[oaicite:1]{index=1}

---

## 📌 Features

- Fully **responsive layout** for all device sizes  
- **Mobile-first design** for optimal performance  
- Navigation menu with dynamic JavaScript behavior  
- Structured sections such as Hero, About, Menu, and Contact
